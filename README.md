# Hawker Culture Website — FED CA2 Project
This is an interactive website created as part of the Front-End Web Development CA2 (Continuous Assessment 2) project for my course.  
**Grade: A**

## Features
- Responsive multi-page layout using HTML5, CSS3, Bootstrap, and JavaScript  
- Interactive form with validation  
- Search/filter functionality to explore content  
- Flip card animations for engaging presentation  
- Thematic design showcasing Singapore’s hawker culture  

## Technologies Used
- HTML5  
- CSS3 (Flexbox, media queries, transitions/animations)  
- JavaScript (DOM manipulation, event handling, validation)  
- Bootstrap 5 (responsive grid system, components, utilities)
